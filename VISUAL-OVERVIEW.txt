🛡️ SEGURANÇA
════════════════════════════════════════════════════════════════════════════

✅ SQL Injection Prevention
   Todas queries usam parameterized queries (?)
   Nunca concatena strings em SQL

✅ Validação de Entrada
   Data, ID, mensagem — tudo validado
   Limite de caracteres (max 500)

✅ Soft Deletes (GDPR Compliant)
   Nunca deleta realmente
   Marca como inativo (ativo = 0)
   Recuperável a qualquer hora

✅ Confirmação para Operações Riscosas
   Deletar tarefa? Pede confirmação
   Timeout 5 minutos
   Rastreado em logs

✅ Auditoria Completa
   Cada ação em logs/acoes.log
   Timestamp, usuário, ação, detalhes
   Sucesso/erro registrado


📊 BANCO DE DADOS
════════════════════════════════════════════════════════════════════════════

SQLite com:
  ✅ WAL mode (Write-Ahead Logging)
  ✅ Índices para performance
  ✅ Migrações automáticas
  ✅ Soft deletes (ativo field)
  ✅ Timestamps automáticos

Tabelas:
  tarefas (id, descricao, materia, tipo, data_vencimento, prioridade, 
           concluida, criado_por, criado_em, concluida_em, ativo)
  
  lembretes (id, mensagem, criado_por, criado_em, ativo)
  
  schema_version (versao, aplicada_em) ← controla migrações

📞 SUPORTE RÁPIDO
════════════════════════════════════════════════════════════════════════════

Erro "ECONNREFUSED"?
  → API não está rodando
  → Execute: node src/api/index.js em outro terminal

Nenhuma tarefa mostrando?
  → Verificar se está marcadaativo=0
  → Ver logs: cat logs/acoes.log

QR Code não aparece?
  → Pode estar acima. Scroll up no terminal

Quer alterar porta?
  → Configure: API_PORT=3001 node start.js
